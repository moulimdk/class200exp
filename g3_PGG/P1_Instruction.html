  • This game is called Public Good Game
  • You are in a group of three.
  • You have an endowment of {{ endowment }} units (Endowment).
  • You can contribute a part of your endowment to a public good fund (Contribution). The rest will remain with you.

Public Good Fund

  • Each member of the group you are in contribute to this fund. It can be zero (0) units.
  • The fund returns {{ multiplier }} times the total contribution. For example: If you contribute 1 unit and the total contribution is 3 units then the fund will return {{ multiplier }}×3 units = {{ pgeTotal }} units.
  • Each of you will receive one-third of the total return. In the given example, you will receive {{ multiplier }} units from the fund. This is called the Public Good Earning.

Total Earning

  • Your Earning = Endowment - Contribution + Public Good Earning
  • Example: Your Earning = {{ endowment }} - 1 + {{ multiplier }} units = {{ example }} units
